# NFL Dashboard - Roadmap & Progress

## Project Overview
A fantasy football decision-making tool that visualizes NFL team strategies and player statistics. Currently using mock data for prototype development. Real data integration planned for next phase.

**Location:** `/Users/josh/Projects/nfl-dashboard`
**Tech Stack:** Next.js 16 + React 19 + TypeScript + Tailwind 4 + Recharts
**Repository:** https://github.com/archer6004/nfl-dashboard
